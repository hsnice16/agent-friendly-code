# 03 · CI workflow

**Status**: done

## Goal

`.github/workflows/ci.yml` that runs on every PR.

## What shipped

- GitHub Actions workflow at `.github/workflows/ci.yml`.
- Triggers on `pull_request` into `main` and `push` to `main`.
- `concurrency` group cancels in-progress runs when a newer commit lands.
- Steps:
  1. `oven-sh/setup-bun@v2` (bun 1.2.16, pinned)
  2. `bun install --frozen-lockfile`
  3. `bun x @biomejs/biome ci .` — lint + format check (see task 04)
  4. `bun x tsc --noEmit` — typecheck
- A test step (`node --test`) will be added once `tasks/0.2.0/01-tests.md` lands.

## Acceptance

- `bun x @biomejs/biome ci .` passes locally → PR build passes.
- `bun x tsc --noEmit` passes locally → PR build passes.
- Workflow runs in under ~90 s on the small current codebase.
